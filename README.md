## **Project Overview**

This repository contains my portfolio as a Student Digital Ambassador. The project demonstrates essential digital skills across five modules, including visual communication, professional networking, technical collaboration, and cybersecurity.

## **Module Summaries**
Module 1: Visual Resource – A digital literacy awareness infographic created to help peers understand digital safety and tools.
Module 2: Digital Portfolio – Foundational professional profiles established on GitHub and other key platforms for academic and career growth.
Module 3: Platforms & Collaboration – Completion of a coding challenge and the creation of a digital literacy quiz for peer engagement.
Module 4: Communication & Etiquette – Professional email drafts for academic/industry scenarios and a social media "Do’s and Don’ts" guide.
Module 5: Cybercrime Awareness – A detailed case study on online threats and a prevention checklist for student safety.
